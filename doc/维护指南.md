# 一、项目简介

SpellingTrainer 是一个使用 **Swift + SwiftUI** 编写的 macOS 单词拼写练习工具。

核心目标：

- 从 Zotero 高亮文本快速导入生词
- 使用类似 QWERTY Learner 的方式进行拼写训练
- 提供轻量、专注、无云依赖的本地学习工具

技术栈：

- Swift
- SwiftUI
- macOS AppKit 运行环境
- 本地 JSON 数据存储
- GitHub Actions 自动构建

# 四、代码结构说明

项目主要包含以下模块。

## 1 App 入口

```
SpellingTrainerApp.swift
```

- 创建主窗口

核心数据结构：

```

```

字段包括：

- 时间信息：`createdAt`、`lastTrainedAt`
- 练习统计：`attemptCount`、`correctCount`、`wrongCount`
- 复习调度：`nextReviewAt`、`intervalDays`、`difficulty`

## 3 数据存储

当前使用：

```
JSON 本地存储
```

文件名：`vocab.json`

存储位置取决于设置：

- 本地：`Application Support/SpellingTrainer/vocab.json`
- iCloud：`iCloud Documents/SpellingTrainer/vocab.json`

`useICloudSync` 切换时会尝试做双向迁移（目标不存在时复制）。

优点：

- 简单
- 易于备份
- 无需数据库

# 五、导入与导出

## 1) CSV 导入

入口：`importCSV(from:mergeMeaning:)`

行为：

- 使用内置 `CSV.parse` 解析
- 识别 header（`word`,`meaning`）并跳过
- 逐行 `upsert`

## 2) Zotero 导入

入口：

- `importZoteroClipboard`
- `importZoteroText`

支持两类模式：

- `“word” ... "meaning"`
- `“word” ... 🔤meaning🔤`

附加能力：

- 自动提取来源引用 `[ ... ]`
- 自动提取 `zotero://...` URL
- 清洗重复词头（例如 meaning 前缀重复了 word）

## 3) 导出

- `exportCSVString()`
- `exportJSONString(pretty: true)`

UI 侧通过 `NSSavePanel` 导出到用户指定路径。

# 六、练习引擎

主要负责：

- 抽取 session 单词
- 统计练习结果
- 更新复习状态


核心类型：`PracticeEngine`

## 1) 会话启动

`start(with:)` 的主要流程：

1. 重置会话状态。
2. 读取设置：
- `maxNewPerSession`（默认 10）
- `maxReviewPerSession`（默认 30）
3. 筛选到期词：`nextReviewAt == nil` 或 `<= now`。
4. 拆分新词/复习词并排序。
5. 取配额后混排成队列。
6. 无到期词时走 maintenance batch（随机抽样）。

## 2) 判题与更新

`submit(store:)`：

- 判定规则：`normalize(input) == normalizedWord`
- 正确：
- `correctCount +1`
- `intervalDays` 倍增（上限 365）
- `difficulty -1`（下限 1）
- 计算下次复习时间
- 错误：
- `wrongCount +1`
- `intervalDays = 1`
- `difficulty +1`（上限 5）
- 同词立即重输（当前题不切换）
- 并加入 recycle 队列延迟再出现

## 3) 模式

- `strict`：默写
- `copy`：跟打（含错误位置提示和 ghost text）

# 七、UI 层

主要使用：

```
SwiftUI
```

主要界面：

- Sidebar
- Practice Panel
- Setting

`RootView` 使用 `NavigationSplitView`：

- 左：`SidebarView`
- 右：`PracticeView`

`SidebarView` 主要入口：

- 添加单词（`+`）
- 搜索词条
- 右键删除
- `Import CSV...`
- `Paste Zotero`
- `Export CSV...`
- `Export JSON...`
- `Merge meaning on import/upsert` 开关

`PracticeView` 主要入口：

- `Start Practice`
- `Stop`
- `显示释义` 开关
- `默写` 开关（strict/copy 切换）

`SettingsView` 主要配置：

- `maxNewPerSession`
- `maxReviewPerSession`
- `Use iCloud Drive sync`
- `Reset to defaults`

# 十二、macOS 安全限制说明

由于当前应用 **未使用 Apple Developer 签名**，第一次运行需要执行：

```
sudo xattr -r -d com.apple.quarantine /Applications/SpellingTrainer.app
```

否则 macOS 可能提示应用无法打开。

# 十三、常见问题


## 1 GitHub Actions build 失败

常见原因：

- deployment target 版本过高
- Xcode SDK 不兼容
- Swift actor 隔离问题

解决方法：

检查 Actions 日志中的第一条 `error:`。


## 2 App 打开被系统拦截

解决方法：

```
右键 → Open
```

或使用：

```
xattr -d
```
